# Corporate Flow Command Center

The design combines the `main` branch visual DNA with a high-density enterprise workspace: near-black graphite shell, warm-neutral canvas, condensed headings, mono system labels, ember identity accent and hairline borders.

Functional colors are teal for actions, violet for workflow, gold for governance, emerald for completion, amber for risk and coral for failure. Cards use 6px radii and restrained shadows. Tables prioritize scanning; controls have visible focus; layouts support 1440 through 390 px, dark/light mode and reduced motion.
